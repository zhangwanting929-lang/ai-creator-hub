# Skills

存放可复用的 Codex 或 Agent Skills。

```text
skill-name/
├── SKILL.md
├── scripts/
├── references/
└── examples/
```

每个 `SKILL.md` 应明确说明适用场景、触发条件、输入、执行流程、输出和边界。

## Available Skills

- [`knowledge-inbox-organizer`](knowledge-inbox-organizer/SKILL.md): 将截图、链接、文件和零散想法整理成可追溯、有关联的 Obsidian Markdown 知识。

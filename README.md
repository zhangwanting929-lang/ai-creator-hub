# AI Creator Hub

张婉婷的 AI 创作、项目实践与可复用能力仓库。

这里持续沉淀 AI 内容生产、电商视觉、工作流自动化和个人知识系统相关的公开项目、Skills、提示词与案例。

## 导航

| 目录 | 内容 |

| [`projects/`](projects/) | 可运行、可展示的 AI 项目 |

| [`content/`](content/) | 公开文章、教程和内容案例 |
| [`prompts/`](prompts/) | 已脱敏、可复用的提示词 |

## 当前方向

- AI 电商视觉与内容生产
- AI 工作流和 Agent Skills
- Codex + Obsidian 个人知识系统
- 面向真实业务问题的 AI 工具实践

## 发布原则

- 只发布已脱敏、具备复用价值的内容。
- 项目必须说明目标、输入、输出和使用方法。
- Skills 必须包含清晰的触发条件、流程和示例。
- 不提交客户原文件、个人隐私、账号凭证或 API 密钥。
